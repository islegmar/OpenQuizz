Help :: Open Quizz

Open Quizz is (another) quizz mobile application with some interesting features:

  • Can be extended with your own questions
  • It can be played alone or with other people without the need of registrations, thanks to Bluetooth

First Steps

You can play in three different modes:
  • Single Player
  • MutiPlayer - Server
  • MutiPlayer - Client
There are also some configurations you need to set before start playing, depending on the mode:
  • Player: this is your name, not needed in Single Player mode.
  • Data URL: this is required unless you play as Client in Multiplayer mode. This is a public JSON with the questions and the answers, which format is described below. You can enter your own URL or chose one of the preset

Multiplayer

Very Important: all the devices must have bluetooth enabled and they must be paired each other. Unfortunatelly the current version of the application (in fact this is a limitation of MIT App Inventor is not capable of doing the pairing so you have to do it by yourself.

One of the devices will act as Server and it must have set:
  • Player's name
  • Data URL
It will the request of the other players until it decides to Start the game
